About the Game

Minesweeper is a classic single-player puzzle game that challenges your logic and reasoning skills. The game takes place on a grid filled with hidden mines. Your goal is to clear all the safe tiles without detonating any mines.

Each time you click a tile, you'll either uncover a mine (and lose), or reveal a number that tells you how many mines are touching that tile. Using these clues, you must carefully mark the locations of all the mines and uncover the rest of the grid to win.

Simple to learn but tricky to master, Minesweeper is a timeless game of deduction and strategy.

Game features

  • Written entirely in C language
  • Text-based console interface
  • Randomized mine placement
  • Numbered tiles show how many mines are adjacent
  • Auto-reveal of safe areas when zero is uncovered
  • Game-over message when stepping on a mine
  • Victory condition when all non-mine tiles are cleared
  • Clean code structure for easy understanding and modification

How to play

Objective
Uncover all the tiles on the grid that do not contain mines.

Rules

  1. Grid Layout:
    • The game is played on a grid of hidden tiles(-) having numbers for rows and columns as coordinates.
    • Some tiles contain hidden mines, and others do not.
  2. Revealing Squares:
    • Click on a tile to reveal what's underneath.
    • If you reveal a mine, you lose the game.
    • If the tile has a number, it tells you how many mines are adjacent to it (including diagonally).
    • If the tile is empty (0 adjacent mines), nearby tiles will automatically be revealed.
  3. Using Numbers:
    • Use the numbers to deduce where the mines are located.
    • For example, a "2" means there are two mines in the surrounding eight tiles.
  4. Winning the Game:
    5. Successfully reveal all non-mine tiles to win.
Tips
  • Start from the corners or edges to open up more of the grid.
  • Use logic and patterns to identify where mines are likely hidden.
